The UnitEase is a web-based platform designed to streamline apartment operations by providing role-based access for Tenants, Workers, and Admins.

This system allows tenants to submit requests, workers to manage assigned tasks, and admins to oversee accounts, announcements, and overall apartment operations.

Authentication System:
- Login and signup for Tenants, Workers, and Admins

Tenant Portal:
- Track payment status
- view announcements
- submit requests

Worker Portal:
- View assigned tasks
- update progress or completion status

Admin Portal:
- Manage tenants and workers
- Post announcements
- Monitor Requests

Database Integration:
- All data stored and managed with SQL and PHP

Technologies Used:
- Frontend: HTML, CSS
- Backend: PHP
- Databse: MySQL/SQL
